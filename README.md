# docker-lab

Este proyecto contiene un microservicio desarrollado en Java 17 con Gradle y empaquetado como contenedor Docker. El microservicio expone un endpoint en el puerto 8085.

## 🛠️ Tecnologías utilizadas

- Java 17 (Temurin)
- Gradle
- Spring Boot
- Docker
- Docker Compose
